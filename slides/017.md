### Configuração e Conexão

<br>

```c
// Função chamada quando uma mensagem MQTT chega
void callback(char* topic, byte* payload, unsigned int length) {
    // Converte o payload para uma string e depois para inteiro
    payload[length] = '\0';
    int velocidade_percent = atoi((char*)payload);

    // Garante que o valor está entre 0 e 100
    if (velocidade_percent >= 0 && velocidade_percent <= 100) {
        // Mapeia de 0-100 para 0-255 (resolução de 8 bits)
        int dutyCycle = map(velocidade_percent, 0, 100, 0, 255);
        analogWrite(ENA_PIN, dutyCycle); // Gera o sinal PWM
    }
}
```
